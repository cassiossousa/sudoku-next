# Sudoku Next

Interactive Sudoku web application built with Next.js and TypeScript.

## Project Summary

- Built a Sudoku puzzle interface using Next.js and React.
- Implemented Sudoku validation logic for checking puzzle correctness.
- Developed responsive UI interactions for puzzle gameplay.

## Tech Stack

### Core

- Next.js
- React
- TypeScript

### Tooling

- ESLint
- Prettier

## Architecture Highlights

- Client-side state management for puzzle board updates.
- Component-based UI architecture using React.
- Puzzle validation logic separated from UI rendering.

## Key Features

- Interactive Sudoku board
- Puzzle validation
- Responsive layout

## Quality

- Type-safe codebase using TypeScript
- Linting with ESLint
- Consistent formatting with Prettier
